Sync Your Cloud

Java App to sync multiple file storage accounts like Dropbox, Google Drive, SkyDrive and Owncloud.

Download this project as a .zip file Download this project as a tar.gz file

Client multiplateforme pour synchroniser vos Cloud Drive comme Dropbox, Google Drive, SkyDrive et Owncloud.

En cours de développement première release prévu dans quelques semaines.

Features

  • Java 1.7
  • Support pour Dropbox, Google Drive, SkyDrive, Owncloud
  • Multiple comptes
  • Synchronisation entre Drive

Évolution

  • Web-service
  • Support WebDAV, Hubic, autres ...

A propos

Projet dans le cadre du projet d'année 2013 à l'ESGI.
